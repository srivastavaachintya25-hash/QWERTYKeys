🎹 QWERTYKeys

A modern virtual piano website that lets users play piano notes directly using their computer keyboard.

🌐 Live Demo:
[QWERTYKeys Live Website](https://qwertykeys.netlify.app/)

✨ Features
🎹 Interactive virtual piano
⌨️ Play using keyboard keys
🖱️ Mouse click support
🎵 Real piano sounds
🌑 Modern dark UI
✨ Key press animations
📱 Responsive design
🎼 Two octave support
🔊 Smooth sound playback
🎙️ Recording & playback system
🔗 Social & contact links
🚀 Deployed online with Netlify
